🏦 Advanced Banking System

A banking system built using Python and MySQL that manages user accounts, transactions, and balance operations. It supports deposits, withdrawals, fund transfers, and balance tracking with proper validation, structured program logic, and persistent data storage.

Features

  • Create and manage user bank accounts
  • Deposit and withdraw money via API endpoints
  • Transfer funds between accounts
  • Balance tracking with validation
  • RESTful API endpoints using FastAPI
  • Interactive API documentation via Swagger UI (/docs)
  • Data validation using Pydantic schemas
  • Database operations using SQLAlchemy ORM

System Operations

  • Create Account → register new user account
  • Deposit → add money to account
  • Withdraw → remove money with validation
  • Transfer → send money between accounts
  • Check Balance → view current balance

Internal Architecture

The system follows a modular structure where each banking operation is handled by specific functions or classes.

  • create_account() → registers new user
  • deposit() → handles money addition
  • withdraw() → processes withdrawals with validation
  • transfer() → manages fund transfers
  • check_balance() → retrieves account balance
